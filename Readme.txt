# Windows_Turn_Off_Display

Passaggi per eseguire direttamente il file .ps1 con doppio clic:

1. Scarica l'eseguibile, spostalo sul desktop

2. Trova l'eseguibile di PowerShell:

   Su Windows, l'eseguibile di PowerShell si trova generalmente in `C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe`.

3. Associa l'estensione .ps1 a PowerShell:

   Fai clic con il tasto destro sul file `SpegniMonitor.ps1`. Seleziona `Apri con -> Scegli un'altra app`. Scorri verso il basso e seleziona `Cerca un'altra app su questo PC`. Naviga fino al percorso `C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe` e selezionalo. Assicurati di selezionare la casella `Usa sempre questa app per aprire i file .ps1`.
